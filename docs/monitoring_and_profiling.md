# Monitoring and Realtime Profiling

## Runtime Monitoring

- Topic latency and loss are reported by `scripts/collect_topic_metrics.sh`.
- Core KPI includes message timeliness and drop ratio.

## Realtime Profiling

- Profiling report skeleton generated by `scripts/profile_realtime.sh`.
- Track callback latency p95/p99 and deadline misses.

## Recommendation

- Integrate metrics exporter (Prometheus/OpenTelemetry).
- Gate merges if KPI regression exceeds threshold.
