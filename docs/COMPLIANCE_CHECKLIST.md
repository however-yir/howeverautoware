# Apache-2.0 Compliance Checklist

## Required Retention

- Keep `LICENSE` text in redistributed artifacts.
- Keep `NOTICE` content when present upstream.
- Mark significant changes with source/date and change summary.

## Release Gate

1. Verify third-party notice table is up to date.
2. Verify license files are preserved in release package.
3. Verify modified files contain change attribution.
