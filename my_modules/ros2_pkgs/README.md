# My Modules (ROS 2 Packages)

This directory contains custom ROS 2 packages with clear dependency boundaries.

- `my_vehicle_perception_bridge`: topic-interface decoupling
- `my_vehicle_planning_strategies`: planning strategy plugin point
- `my_vehicle_control_safety`: command saturation and fallback

Naming follows ROS 2 package conventions (`lowercase_with_underscores`).
